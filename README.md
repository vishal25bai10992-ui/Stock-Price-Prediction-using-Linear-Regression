# Stock-Price-Prediction-using-Linear-Regression
This project is a beginner-friendly implementation of stock price prediction using Machine Learning. It uses historical stock data fetched from the Yahoo Finance API and applies a Linear Regression model to predict the next day’s closing price based on the current day’s price.

Features
Fetch real-time stock data using Yahoo Finance Predict next day's closing price Visualize actual vs predicted prices Simple and easy-to-understand code

Tech Stack
Python 🐍 Pandas NumPy Matplotlib Scikit-learn yFinance
